  • Formularios, tablas, etc.

En HTML, los formularios y tablas son estructuras fundamentales que permiten a los usuarios interactuar con una página web y organizar datos de manera clara.

  • Formularios (<'form>)

    Los formularios son esenciales para recolectar datos del usuario, como información de contacto, comentarios, o datos de inicio de sesión. Los formularios están compuestos por varios elementos de entrada (inputs), botones, y otros componentes.
    Estructura básica:

    <'form action="procesar_formulario.php" method="post"><'/form>

    Atributos principales:

    • 'action': Especifica la URL a la que se enviarán los datos del formulario.
    • 'method': Define el método HTTP para enviar los datos (GET o POST).
    • Campos de Entrada (<'input'>)

      La etiqueta <'input'> es utilizada para crear diversos tipos de campos de entrada, como texto, contraseñas, casillas de verificación, etc.
      Tipos comunes de <'input'>:

      • Texto (type="text"):

        • <'label for="nombre">Nombre:<'/label>
        <'input type="text" id="nombre" name="nombre">

        Resultado:

      • Contraseña (type="password"):

        • <'label for="password">Contraseña:<'/label>
        <'input type="password" id="password" name="password">

        Resultado:

      • Correo electrónico (type="email"):

        • <'label for="email">Correo electrónico:<'/label>
        <'input type="email" id="email" name="email">

      • Casilla de verificación (type="checkbox"):

        • <'label for="suscribir">Suscribirse:<'/label>
        <'input type="checkbox" id="suscribir" name="suscribir">

      • Botón de envío (type="submit"):

        • <'input type="submit" value="Enviar">

    • Área de Texto (<'textarea>)

      • La etiqueta <'textarea> se utiliza para crear un campo de texto multilínea.
      Ejemplo:

      <'label for="mensaje">Mensaje:<'/label>
      <'textarea id="mensaje" name="mensaje" rows="4" cols="50"><'/textarea>

      Resultado:

    • Listas de selección (<'select> y <'option>)

      • Para crear menús desplegables.
      Ejemplo:

      <'label for="pais">País:<'/label>
      <'select id="pais" name="pais">
      <'option value="mexico">México<'/option>
      <'option value="espana">España<'/option>
      <'option value="argentina">Argentina<'/option>
      <'/select>
