En HTML, las acciones multimedia como la inserción de imágenes, videos, y otros tipos de contenido multimedia se logran mediante el uso de etiquetas específicas.
  • Imágenes

    La etiqueta <"img> se utiliza para insertar imágenes en una página web. Es una etiqueta auto-contenida, lo que significa que no necesita una etiqueta de cierre.
    Atributos Principales:

    • "src": Define la ruta o URL de la imagen.
    • "alt": Proporciona un texto alternativo que se muestra si la imagen no se carga.
    • "width" y "height": Controlan las dimensiones de la imagen.

    Ejemplo: <"img src="imagen.jpg" alt="Descripción de la imagen" width="500" height="300">

    Este código inserta una imagen de 500 píxeles de ancho y 300 píxeles de alto con un texto alternativo "Descripción de la imagen".

    Tambien puedes poner imagenes en movimiento o tambien llamados "Marquesinas" con el siguiente codigo:

    <"marquee direction=left><"img src="imagen.jpg"><"/marquee>
    Ya dependera de si quieres la imagen valla a la Derecha (right), Izquierda (left), arriba (up) o abajo (down)

  • Videos

    La etiqueta <"video> se usa para insertar videos en una página web. A diferencia de <"img>, esta etiqueta requiere una etiqueta de cierre y puede contener varios elementos fuente.

    Atributos principales:

    • "src": Define la ruta o URL del video.
    • "controls": Agrega controles de reproducción (play, pause, volumen, etc.).
    • "autoplay": Hace que el video se reproduzca automáticamente cuando la página se carga.
    • "loop": Reproduce el video en bucle.
    • "muted": Silencia el video de manera predeterminada.

    Ejemplo básico:
    <"video src="video.mp4" controls width="600" height="400"><"/video>

  • Audio (<"audio>)

    La etiqueta <"audio> se usa para insertar archivos de audio en una página web.

    Atributos principales:

    • "Osrc": Define la ruta o URL del archivo de audio.so
    • "controls": Agrega controles de reproducción (play, pause, volumen, etc.).
    • "autoplay": Hace que el audio se reproduzca automáticamente cuando la página se carga.
    • "loop": Reproduce el audio en bucle.
    • "muted": Silencia el audio de manera predeterminada.

    Ejemplo básico:

    <"audio src="audio.mp3" controls><"/audio>
