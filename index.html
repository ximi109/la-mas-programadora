¿HTML?

HTML es el lenguaje de marcado estándar utilizado para crear y estructurar contenido en la web. Es el esqueleto de una página web, proporcionando la estructura básica mediante etiquetas que definen elementos como títulos, párrafos, enlaces, imágenes, y más.HT
Componentes de HTML

Los componentes básicos de HTML son las etiquetas (o elementos), atributos, y el contenido que conforman la estructura de una página web.
  • Etiquetas :) |1|

    Las etiquetas son los bloques de construcción fundamentales de HTML. Se utilizan para definir elementos en un documento HTML. La mayoría de las etiquetas vienen en pares: una etiqueta de apertura y una etiqueta de cierre. La etiqueta de apertura indica el inicio de un elemento, mientras que la etiqueta de cierre marca su final

    • Etiqueta de apertura: <"elemento>
    • Etiqueta de cierre: <"/elemento>

      • Ejemplo:

      Aquí, <"p> es la etiqueta de apertura y <"/p> es la etiqueta de cierre

  • Atributos

    Los atributos proporcionan información adicional sobre los elementos HTML. Se colocan dentro de la etiqueta de apertura y generalmente consisten en un nombre y un valor.

    <"img src="imagen.jpg" alt="Descripción de la imagen">

    En este ejemplo, "src" es un atributo que especifica la fuente de la imagen, y "alt" proporciona un texto alternativo en caso de que la imagen no se pueda mostrar.

  • Contenido

    El contenido es el texto o el medio que se encuentra entre una etiqueta de apertura y una etiqueta de cierre. Este contenido es lo que se mostrará al usuario en la página web.

    <"h1>Este es un título<"/h1>

    En este ejemplo, "Este es un título" es el contenido que se mostrará como un encabezado.

  • Estructura básica de un documento HTML

    Un documento HTML tiene una estructura básica que incluye las siguientes secciones:

    • <"!DOCTYPE html>: Declara el tipo de documento y la versión de HTML utilizada.
    • <"html>: El elemento raíz que contiene todo el contenido de la página web.
    • <"head>: Contiene metadatos sobre el documento, como el título y enlaces a hojas de estilo.
    • <"title>: Define el título de la página que aparece en la pestaña del navegador.
    • <"body>: Contiene el contenido visible de la página, como texto, imágenes, y enlaces.
